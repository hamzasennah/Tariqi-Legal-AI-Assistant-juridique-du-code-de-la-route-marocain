# Sources officielles et politique de confiance

Le projet doit répondre uniquement depuis des sources institutionnelles marocaines.

## Sources A+

- Secrétariat Général du Gouvernement : textes consolidés, loi n° 52-05, décrets.
- Bulletin Officiel : référence juridique opposable.
- Ministère du Transport et de la Logistique : textes réglementaires publiés.
- NARSA / Khadamat NARSA : pages pratiques et tableaux officiels.

## Sources A

- Pages Khadamat NARSA sur les infractions, le paiement, la déclaration, la réclamation et le permis à points.
- Plateformes de services publics liées aux démarches et paiements.

## Sources B

- Instructions institutionnelles de signalisation routière.
- Rapports statistiques officiels NARSA.

## Sources exclues comme base principale

- blogs privés ;
- forums ;
- posts de réseaux sociaux ;
- vidéos non institutionnelles ;
- PDF sans origine officielle ;
- résumés d'auto-écoles ;
- copies non traçables.

## Règle de réponse

Si une information n'est pas présente dans les passages retrouvés, l'assistant doit le dire clairement et proposer de consulter l'autorité compétente ou le texte officiel.

## Références suivies dans le manifeste

Le fichier `data/raw/sources_manifest.json` contient les URL et métadonnées utilisées par le pipeline.
